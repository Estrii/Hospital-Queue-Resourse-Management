\# Simulimi i Radhës dhe Menaxhimi i Burimeve në Spital



\## Përshkrimi

Ky projekt në Java simulon një sistem spitalor që menaxhon:

\- Radhët e pacientëve duke përdorur prioritete sipas triage

\- Burimet kritike (mjekë, infermierë, dhoma)

\- Faza të ndryshme të trajtimit të pacientëve

\- Menaxhimin e emergjencave dhe mbingarkesën e sistemit

\- Llogaritjen e kohës së pritjes dhe statistikat

\- Ruajtjen dhe rikthimin e gjendjes së simulimit



\## Karakteristikat

\- Radhë prioritare dinamike (triage)

\- Alokim burimesh në kohë reale

\- Menaxhimi i ciklit të jetës së pacientëve

\- Statistika për performancën e spitalit

\- Ruajtje dhe rikthim i gjendjes së simulimit



\## Teknologjitë

\- Java

\- Parimet e OOP (Object-Oriented Programming)

\- PriorityQueue

\- Serialization

\- Git \& GitHub
